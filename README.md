# Air
Очиститель воздуха - "Очиститель"
8 вентиляторов 120мм

- Очиститель состоит из 3х деталей
  - Блок выдува
  - Блок фильтра
  - Блок вдува
- Блок выдува 
  - Гнездо для питания
  - 4 вентилятора
- Блок фильтра
  - Этот блок нужен, чтобы не менять "блок вдува" и "блок выдува", а изменять только "блок фильтра". Если будет необходимость конструктивно изменить место под фильтр, количество фильтров и тд.
  - Фильтр вставляется в "пазы"
  - Фильтр прижимается сверху пластиной
- Блок вдува
  - 4 вентилятора. На каждой стороне по 1 вентилятору
  - Перед ними возможность установить "фильтр сетку" на магнитах. Предусмотреть что-то магнитное???
