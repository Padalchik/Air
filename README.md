# Air
Очиститель воздуха - "Очиститель"
8 вентиляторов 120мм

- Очиститель состоит из 3х деталей
  - Блок выдува
  - Блок фильтра
  - Блок вдува
- Блок выдува 
  - Гнездо для питания
  - 4 вентилятора
    - Над вентиляторами предусмотреть что-то (пластмассовую деталь на всю плоскость выдува) для защиты от попадания крупных предметов в вентиляторы.
  - Регулятор оборотов
    - Находится на "скосе" корпуса
  - Материнская плата
  - Должен быть уплотнитель по периметру, чтобы исключить забор воздуха. Воздух должен идти только через фильтр
- Блок фильтра
  - Этот блок нужен, чтобы не менять "блок вдува" и "блок выдува", а изменять только "блок фильтра". Если будет необходимость конструктивно изменить место под фильтр, количество фильтров и тд.
  - Фильтр вставляется в "пазы"
  - Фильтр прижимается сверху пластиной
- Блок вдува
  - 4 вентилятора. На каждой стороне по 1 вентилятору
  - Перед ними возможность установить "фильтр сетку" на магнитах. Предусмотреть паз/выемку? Предусмотреть что-то магнитное???
